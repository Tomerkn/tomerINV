# מדריך תיקון גרסאות Flask ו-Werkzeug

## 🎯 הבעיה
VSCode דורש גרסאות ספציפיות של Flask ו-Werkzeug שצריכות להיות תואמות זו לזו.

## 🔧 הפתרון - שימוש בסקריפטים

### 1. בדיקה מהירה
```bash
python check_versions.py
```

**פלט מוצלח:**
```
Flask: 2.2.5
Werkzeug: 2.2.3
✅ תואם
```

### 2. תיקון מלא (אם יש בעיה)
```bash
python fix_versions.py
```

**מה הסקריפט עושה:**
- בודק תאימות גרסאות נוכחיות
- מתקן אוטומטית גרסאות לא תואמות
- מעדכן requirements.txt
- בודק שהאפליקציה עובדת
- יוצר קובץ נעילת גרסאות

## 📋 הגרסאות המומלצות (יציבות מקסימלית)
- Flask==2.2.5
- Werkzeug==2.2.3
- Flask-WTF==1.1.1
- WTForms==3.0.1
- Flask-Login==0.6.3

## ⚡ שימוש יומיומי

**לפני כל עבודה על הפרויקט:**
```bash
python check_versions.py
```

**אם יש בעיה:**
```bash
python fix_versions.py
```

## 🔒 מניעת בעיות עתידיות

1. **אל תעדכן Flask ידנית** - השתמש בסקריפטים
2. **הרץ בדיקה לפני כל פיתוח**
3. **השתמש בגרסאות הנעולות ב-requirements.txt**

## 🚨 אם משהו לא עובד

1. ודא שאתה בתיקייה הנכונה
2. ודא שהסביבה הוירטואלית פעילה
3. הרץ: `python fix_versions.py`
4. אם עדיין לא עובד, התקן מחדש:
   ```bash
   pip install -r requirements.txt --force-reinstall
   ```
